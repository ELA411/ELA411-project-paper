\subsection{BCI Requirements}
\textcolor{violet}{The complete requirements for the BCI system:
\begin{enumerate}
    \item EEG
    \begin{enumerate}
        \item Record signal
        \begin{enumerate}
            \item Time stamp signal
        \end{enumerate}
        \item Stream signal to a PC
        \begin{enumerate}
            \item Store for offline analysis
            \item Display data lost/drop
            \item Display data quality
            \item Visualize signal
        \end{enumerate}
        \item Filter signal (reduce noise and artifacts)
        \item Apply feature extraction on the signal
        \item Use AI to decode navigation variables from signal
        \begin{enumerate}
            \item Display decoding accuracies
        \end{enumerate}
    \end{enumerate}
    \item EMG
    \begin{enumerate}
        \item Record signal
        \begin{enumerate}
            \item Time stamp signal
        \end{enumerate}
        \item Stream signal to a PC
        \begin{enumerate}
            \item Store for offline analysis
            \item Display data loss
            \item Display data quality
            \item Visualize signal
        \end{enumerate}
        \item Filter signal (reduce noise and artifacts)
        \item Apply feature extraction on the signal
        \item Use AI to decode navigation variables from signal
        \begin{enumerate}
            \item Display decoding accuracies
        \end{enumerate}
    \end{enumerate}
    \item Control a Turtlebot3 in Gazebo
    \begin{enumerate}
        \item Using ROS2
        \item Based on EEG
        \begin{enumerate}
            \item No physical movements allowed during control with EEG
            \item Drive
            \item Do not drive
        \end{enumerate}
        \item Based on EMG
        \begin{enumerate}
            \item Turn left
            \item Turn right
        \end{enumerate}
    \end{enumerate}
    \item System should work for various users
    \item System should work in real-time (online) with 1s deadline
    \item Display time delay (response time) from obtaining signal to controlling the Turtlebot3
    \item validate BCI system using a statistical measure
\end{enumerate}}
